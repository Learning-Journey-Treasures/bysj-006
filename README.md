# 基于SSM的酒店客房管理系统+毕业设计示例参考

#### 介绍
技术栈+工具：SSM + MySQL + IDEA2022 + Maven + JSP

系统功能：登录注册、客房预订、公告查看、留言功能、客房管理、入驻管理、退房管理等

系统角色：管理员、酒店前台、普通用户
